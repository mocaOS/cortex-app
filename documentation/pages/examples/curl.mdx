Complete cURL examples for all major MOCA Library API operations. These examples are ready to copy and paste.

## Setup

Set your API configuration as environment variables:

```bash
export MOCA_URL="http://localhost:8000"
export MOCA_API_KEY="your-api-key"
```

## Health & Statistics

### Health Check

```bash
curl "$MOCA_URL/health"
```

### Get Statistics

```bash
curl "$MOCA_URL/api/stats" \
  -H "X-API-Key: $MOCA_API_KEY"
```

## Documents

### Upload a Document

```bash
# collection_id and start_processing are query parameters
curl -X POST "$MOCA_URL/api/upload?collection_id=default&start_processing=true" \
  -H "X-API-Key: $MOCA_API_KEY" \
  -F "file=@document.pdf"
```

### Upload Without Processing (for bulk)

```bash
# start_processing is a query parameter
curl -X POST "$MOCA_URL/api/upload?start_processing=false" \
  -H "X-API-Key: $MOCA_API_KEY" \
  -F "file=@document.pdf"
```

### Process Pending Documents

```bash
curl -X POST "$MOCA_URL/api/documents/process-pending" \
  -H "X-API-Key: $MOCA_API_KEY"
```

### List Documents

```bash
curl "$MOCA_URL/api/documents" \
  -H "X-API-Key: $MOCA_API_KEY"
```

### List Documents in a Collection

```bash
curl "$MOCA_URL/api/documents?collection_id=research" \
  -H "X-API-Key: $MOCA_API_KEY"
```

### Get Document Details

```bash
curl "$MOCA_URL/api/documents/doc_abc123" \
  -H "X-API-Key: $MOCA_API_KEY"
```

### Delete a Document

```bash
curl -X DELETE "$MOCA_URL/api/documents/doc_abc123" \
  -H "X-API-Key: $MOCA_API_KEY"
```

### Reprocess a Document

```bash
curl -X POST "$MOCA_URL/api/documents/doc_abc123/reprocess" \
  -H "X-API-Key: $MOCA_API_KEY"
```

### Bulk Delete

```bash
curl -X POST "$MOCA_URL/api/documents/bulk-delete" \
  -H "X-API-Key: $MOCA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"document_ids": ["doc_1", "doc_2", "doc_3"]}'
```

## Ask AI (RAG)

### Ask a Question

```bash
curl -X POST "$MOCA_URL/api/rag" \
  -H "X-API-Key: $MOCA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "What are the main findings?"}'
```

### Ask About a Collection

```bash
curl -X POST "$MOCA_URL/api/rag" \
  -H "X-API-Key: $MOCA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Summarize the key points",
    "collection_id": "research"
  }'
```

### Fast Mode

```bash
curl -X POST "$MOCA_URL/api/rag" \
  -H "X-API-Key: $MOCA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "What is the company mission?",
    "fast_mode": true
  }'
```

### Streaming Response

```bash
curl -X POST "$MOCA_URL/api/rag/stream" \
  -H "X-API-Key: $MOCA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "Explain GraphRAG in detail"}' \
  --no-buffer
```

### Agentic Mode

```bash
curl -X POST "$MOCA_URL/api/rag/agentic" \
  -H "X-API-Key: $MOCA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Compare the methodologies in papers A and B",
    "enable_reasoning": true
  }'
```

### With Conversation History

```bash
curl -X POST "$MOCA_URL/api/rag" \
  -H "X-API-Key: $MOCA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Can you elaborate on the third point?",
    "conversation_history": [
      {"role": "user", "content": "What are the benefits?"},
      {"role": "assistant", "content": "The main benefits are: 1) ..."}
    ]
  }'
```

## Turbo Mode

### Check Turbo Status

```bash
curl "$MOCA_URL/api/turbo/status" \
  -H "X-API-Key: $MOCA_API_KEY"
```

### Start Turbo Job

```bash
curl -X POST "$MOCA_URL/api/turbo/start" \
  -H "X-API-Key: $MOCA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"runtime_seconds": 3600}'
```

### Stop Turbo Job

```bash
curl -X POST "$MOCA_URL/api/turbo/stop" \
  -H "X-API-Key: $MOCA_API_KEY"
```

### Ask with Turbo

```bash
curl -X POST "$MOCA_URL/api/turbo/ask" \
  -H "X-API-Key: $MOCA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "Summarize the findings"}'
```
